# Matematična fizika I

Projekt pri predmetu Matematična fizika I. 

V poročilu je obravnavan izračun deformacije gumirane jeklene kroglice pod vplivom zunanjega tlaka. Zraven je dodan tudi Python skript `simulator.py`, ki omogoča interaktivno vizualizacijo deformacij pri različnih parametrih.
